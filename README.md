# Django-Ecommerce-app-with-Paystack-Integrated

In this project, we utilized the concepts of python and the django framework to create an ecommerce app which integrates paystack as its means of payment.
For clarity the projects is subdivided into 4 parts:
#Ecom
#Store
#Static
#Database -SQLite

The Ecom folder contains the Project configuration files and the base URL
The Store folder contains the project files, Models, View, and Templates and the projects URL files.
The Static folder contains the CSS files, The Javascripts file and the Images used throughout the project
The database holds sensitive information about the projects Schema
